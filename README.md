# Памятка реаниматолога при отсутствии психиатра

Веб-версия памятки с алгоритмами действий врача-реаниматолога:

- Отравление психоактивными веществами
- Синдром отмены (алкогольный)
- Делирий пожилых
- Деменция

## Открыть онлайн

Страница доступна через GitHub Pages — см. раздел *About* → *Website* или
[перейти по ссылке](./index.html).

## Использование локально

Достаточно открыть `index.html` в любом современном браузере.

---

> Памятка носит справочный характер и не заменяет клинических рекомендаций
> и решения лечащего врача. Дозы и алгоритмы должны быть адаптированы к конкретной
> клинической ситуации, сопутствующей патологии и актуальным инструкциям к препаратам.
